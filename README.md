Jogo do Número Secreto

Um jogo de adivinhação divertido e interativo desenvolvido com JavaScript, onde o jogador tenta descobrir um número gerado aleatoriamente dentro de um intervalo pré-definido.

Como Jogar

O jogo gera um número aleatório dentro de um intervalo de 1 a 10

O jogador insere um palpite.

O jogo informa se o número secreto é maior ou menor do que o palpite.

O jogador continua tentando até acertar o número correto.

Tecnologias Utilizadas

JavaScript

HTML

CSS
